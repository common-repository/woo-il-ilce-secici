WooCommerce ödeme sayfasında Türkiye için il ilçe seçici ekleyecektir.

== Description ==
TemaNinja tarafından hazırlanan WooCommerce il ilçe seçici eklentisi WP City Select eklentisi baz alınarak hazırlanmıştır. Eklentimiz otomatik olarak ülkemizde yer alan tüm il ve bu illere bağlı ilçeleri seçilebilir halde ödeme sayfanıza ekleyecektir.

== Installation ==

Eklentinin kurulumu için şu adımları izlemelisiniz


1. `woocommerce-il-ilce.zip` dosyasını `/wp-content/plugins/` dizinine yükleyin
2. Wordpress admin panelinizdeki 'Eklentiler' menusünden eklentiyi aktive edin
